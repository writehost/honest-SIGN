# Honest Sign

Программы и инструменты для работы с системой маркировки **Честный Знак (ГИС МТ)**.

## 📦 О проекте

Данный репозиторий содержит решения для интеграции с системой Честный Знак, включая:

- работу с кодами маркировки
- отслеживание жизненного цикла кодов
- интеграцию с производственными линиями
- взаимодействие с API ГИС МТ

---

## 🚀 Возможности

- 📡 Получение кодов из Честного Знака (OMS / ГИС МТ)
- 🔍 Отслеживание статусов кодов (эмиссия, ввод в оборот, выбытие)
- 🏭 Интеграция с производственными линиями
- 🖨 Поддержка принтеров и аппликаторов
- 📊 Формирование отчетов (агрегация, выбытие, утилизация)
- 🔗 Работа с ERP / MES / SCADA системами

---

## 💼 Коммерческая версия

Существует **коммерческая версия программного решения**, которая включает:

- полноценную интеграцию с Честным Знаком
- автоматическое отслеживание кодов
- обработку событий в реальном времени
- поддержку агрегации (короб / паллета)
- интеграцию с оборудованием (принтеры, сканеры, камеры)
- расширенную аналитику и отчеты

> 📌 Коммерческая версия предназначена для промышленного использования на производственных линиях.

---

## ⚙️ Технологии

- Backend: .NET / ASP.NET Core
- Интеграция: REST API (Честный Знак, OMS)
- Базы данных: PostgreSQL / MSSQL
- Frontend: Web UI / HMI / SCADA
- Оборудование: принтер-аппликаторы, камеры, сканеры

---

## 📚 Использование

Проект может использоваться для:

- интеграции маркировки в производство
- разработки собственных решений под Честный Знак
- тестирования кодов маркировки
- автоматизации отчетности

Для работы с системой Честный Знак требуется:

- доступ к ГИС МТ

- корректная настройка интеграции (OMS / API)
